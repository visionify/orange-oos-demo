
beverages - v1 2022-03-24 1:52am
==============================

This dataset was exported via roboflow.ai on March 23, 2022 at 8:28 PM GMT

It includes 14768 images.
Beverages are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)
* Resize to 640x640 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* Random rotation of between -2 and +2 degrees
* Random brigthness adjustment of between -3 and +3 percent
* Random Gaussian blur of between 0 and 0.5 pixels
